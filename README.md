# cstyle
Common style for wordpress


## Installation 

In "footer.php" :
`<style><?php echo cstyle(); ?></style>`

Enjoy !

## Usage

Php method : 
`cstyle('.awesome', 'font-family: "Comic Sans MS", cursive, sans-serif;');`

Shortcode method : 
`[cstyle select=".awesome" style="font-family: 'Comic Sans MS', cursive, sans-serif;"]`
